¿Que es GitHub Copilot?

GitHub Copilot es una herramienta de inteligencia artificial basada en la nube desarrollada por GitHub y OpenAI para ayudar y asistir a los usuarios de Visual Studio Code, Visual Studio, Neovim, Eclipse y los entornos de desarrollo integrado (IDE) de JetBrains mediante el autocompletado de código
